Matplotlib: Improving Specifying Colors

Feb. 11, 2021

In this section, Matplotlib documentation explains compatible formats for colors. These inputs are fundamental in configuing data visuals. With more streamlined information on usage and straightforward examples, users of all experience levels benefit from these improvements.

One of the challenges of parsing through the original document was differentiating the examples from the explanation. The bullet points only served to separate each type of format accepted. In the text, commas, parentheses, and phrasing connected each type of format to its respective example. From a visual perspective, this had not been an effective way to communicate these important elements.

I approached this project with the user in mind. I wanted to make sure anyone, ranging from a novice to an experienced developer, could scan the document and get to their respective needs. I saw opportunity to segment blocks of text into tables and charts as well as give additional examples.

Before After

As a technical writer contributing to the library, I modified the content for improved readability. I incorporated a reStructuredText ASCII table to organize the content. This reduced clutter in prose and separated the example from the label. In addition, having a table is a visual improvement to the aesthetic of the document.

I also coded an example of how transparency works. The original document relied on explanatory text to demonstrate the equation for Alpha rendering. There were no example visual aids. I generated a static image to display the properties and respective effects of changing variables. The newly added table compares Alpha with different values.

Several developers praised the changes when the pull request came through. The positive feedback warranted a quick merge for the documentation. Working with Matplotlib has been a true pleasure and learning experience. In combing through the documentation on my own to learn more about the library, I’m excited at each opportunity to improve both my knowledge and the content.
